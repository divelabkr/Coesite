### 무엇을 했나
- 결론: `@coesite/api`에 NestJS Express 플랫폼 어댑터를 추가해 API 부팅 BLOCKER를 제거했다.
- `packages/api/package.json`에 `@nestjs/platform-express@10.4.15`를 추가했다.

### 왜

  - `dependencies`에 `@nestjs/platform-express: 10.4.15` 추가.

  - 신규 의존성 및 전이 의존성 잠금 정보 갱신.
